### MVVM

<img width=400 src="./imageFiles/02.jpg">

- Model
    - UI independent
    - Data + Logic

- View
    - Reflects the Model
    - Stateless, Immutable
    - Reactive
    - Subscribing to what the ViewModel is publishing

- ViewModel
    - Binds View to Model
    - Interpreter, Gatekeeper
    - Doesn't have any connections to Views
