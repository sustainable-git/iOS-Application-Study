## Xcode 단축키
|Commad|Action|
|:-|:-|
|CMD + R|Run|
|CMD + 0~9|Navigator tap|
|CMD + OPT + 0~3|Inspectors tap|
|CMD + SHIFT + Y|Debugger & Console|
|CMD + SHIFT + O|Quick open|
|CMD + /|Comment out|
|CTRL + I|Re-indent|
